# Reference — исходный комплект ПК «ОКО»

Файлы оригинального комплекса ОКО (не коммитятся в git: `*.mdb`, `*.mde`).

| Файл | Назначение |
|------|------------|
| `OKO26-1.mde` | Приложение Access (UI, VBA) |
| `z261.mdb` | База: формы, правила, справочники |
| `ini20261.ini` | Конфигурация |
| `txt2026-1KV.TXT` | Руководство пользователя |
| `docs/` | Документация проекта |

Скопируйте комплект из дистрибутива ОКО в эту папку. Пароль MDB: `12345`.

Генерация схем и выгрузка данных:

```bash
python scripts/generate_schemas_from_mdb.py
python scripts/export_mdb_data.py
```
