## 1. Что такое ПК «ОКО»

**ПК «ОКО»** — программный комплекс корпоративной специализированной бухгалтерской отчётности для организаций группы ПАО «Газпром».

- Ежеквартальная отчётность (1 кв., полугодие, 9 мес., год)
- Суммы в **тысячах рублей**
- Разработчик/куратор: Управление 918/1 Департамента 918 ПАО «Газпром», `METODOL@adm.gazprom.ru`
- Технологии: **Microsoft Access** (`.mde` + `.mdb`), требуется **32-битный MS Access** (Office 2000–2013)

## 5. Архитектура системы

```
OKO26-1.mde          z261.mdb
(логика + UI)   <->  (данные + метаданные + правила)
     |                    |
     +- 126+ форм frm_*   +- 76 таблиц отчётных форм (N01_1, N05_1…)
     +- VBA (скомпил.)    +- a_stblROWs, a_stblFIELDs — строки/графы
     +- Recalc, Aggr…     +- a_tblchecks — 3600 правил увязок
     +- Import/Export     +- sp_kontr, refs — справочники
                          +- tblExcelExport — маппинг в Excel
```

Ключевые идентификаторы в данных:
- **ZID** — идентификатор организации/набора
- **EID** — идентификатор периода/подразделения
- **Number** — номер строки формы

## 8. Бизнес-процессы (из руководства)

1. **Первый запуск:** Администрирование -> выбор организации и подразделения -> «Завести пустые формы»
2. **Перенос сальdo** (до заполнения, 2 этапа):
   - из предыдущего периода -> графы «на начало периода»
   - из аналогичного периода прошлого года -> графы «за аналог. период прошлого года»
3. **Заполнение форм** с расшифровками контрагентов (пороги: 1 тыс. / 5 млн / 50 млн руб.)
4. **Проверка:** Администрирование -> Проверка форм
5. **Сохранение:** Администрирование -> Сохранить на диск -> отправка в головную организацию
6. **Совместная работа:** частичный приём, агрегация, сетевой режим (общая .mdb на сети)

## 10. Полезные SQL-запросы для исследования

```sql
-- Все активные проверки
SELECT Number, LExpCheck, MessageText FROM a_tblchecks WHERE aktiv=True ORDER BY Number;

-- Строки формы N01_1
SELECT RowNo, RowName, Name, A, B, Sort FROM a_stblROWs WHERE TName='N01_1' ORDER BY Sort;

-- Графы формы N01_1
SELECT FName, Ftype, FWidth, FTotal FROM a_stblFIELDs WHERE TName='N01_1';

-- Маппинг Excel для формы
SELECT ExcelSheetName, ExcelRow, ExcelColumn, FormName, FormRow, FormColumn
FROM tblExcelExport WHERE FormName='N01_1';

-- Контрагенты
SELECT id, RowName, OrgForm, inn, kpp FROM sp_kontr;

-- Настройки
SELECT ConfigName, ConfigValue FROM tblConfig;
```
