# Часть 2. Инструкция пользователя организации

## 2.1. Вход в систему

1. Откройте адрес портала в браузере.
2. На главной странице введите **логин** и **пароль**, полученные от администратора.
3. Нажмите **Войти**.

После входа откроется раздел **Мои формы** — ваш личный кабинет.

Внизу бокового меню отображаются ваше имя и название организации.

### Выход
Нажмите **Выйти** внизу бокового меню.

---

## 2.2. Что доступно пользователю организации

### Доступно
| Раздел | Назначение |
|--------|------------|
| **Мои формы** | Ваши заполненные формы |
| **Комплект** | Полнота комплекта вашей организации |
| **Каталог** | Создание отдельных форм |
| **Настройки** | Реквизиты по умолчанию |

### Недоступно
- **Сводка и импорт** — пакетные проверки и агрегация
- Все разделы **Редакторы** и **Администрирование**
- Создание организаций и периодов
- Просмотр и редактирование форм других организаций
- Редактирование сданных форм

Вы видите и работаете **только с формами своей организации** (ваш ZID).

---

## 2.3. Начало работы

### Шаг 1. Проверьте комплект
1. Откройте **Комплект**.
2. Организация уже выбрана (изменить нельзя, если у вас одна организация).
3. Выберите **отчётный период** из списка.
4. Убедитесь, что комплект заведён (прогресс-бар показывает 76/76 или близко к этому).

Если форм не хватает — нажмите **Завести пустые формы (комплект)**. Будут созданы пустые экземпляры для недостающих шаблонов.

### Шаг 2. Откройте форму для заполнения
**Вариант А — из комплекта:**
- На странице **Комплект** в списке заведённых форм нажмите на нужную.

**Вариант Б — из «Мои формы»:**
- **Мои формы** → найдите форму по поиску или фильтру → **Открыть**.

**Вариант В — создать новую из каталога:**
- **Каталог** → найдите шаблон → **Создать**.

---

## 2.4. Заполнение формы

### Реквизиты
В верхней части формы проверьте:
- **Организация** — должна соответствовать вашей;
- **Период** — начало и конец отчётного периода;
- **Код предприятия**, **единица измерения**.

### Таблица
- Вводите данные в ячейки таблицы (как в оригинальных PDF-формах).
- Итоговые строки могут пересчитываться автоматически (если включён **Автопересчёт**).
- Для ручного пересчёта нажмите **Пересчитать**.

### Сохранение
Нажимайте **Сохранить** регулярно — данные записываются на сервер. Несохранённые изменения при закрытии вкладки могут быть потеряны.

### Проверка перед сдачей
- **Проверить увязки** — покажет ошибки в соотношениях между формами;
- **Проверить расшифровки** — для форм с контрагентами.

Исправьте все ошибки (красные) перед сдачей.

### Сдача формы
1. Убедитесь, что форма сохранена.
2. Нажмите **Сдать**.
3. Подтвердите действие.

После сдачи форма переходит в статус **«Сдано»** и **больше не редактируется**. Для исправлений обратитесь к администратору — он вернёт форму в черновик.

### Выгрузка
- **PDF** — печатная версия формы;
- **Excel** — табличная выгрузка;
- **Экспорт JSON** — резервная копия данных формы.

---

## 2.5. Раздел «Мои формы»

Список всех ваших сохранённых форм.

### Поиск и фильтры
- Поиск по названию, коду формы, организации.
- Фильтр по типу формы (N01_1, N02_1 и т.д.).
- Фильтр по статусу: черновики / сдано.

### Удаление
- **Одна форма** — кнопка **Удалить** на карточке.
- **Несколько форм** — отметьте чекбоксами → **Удалить выбранные**.

> Удаление безвозвратно. Не удаляйте сданные формы без согласования с администратором.

### Импорт
Кнопка **Импорт JSON** — загрузка ранее экспортированного экземпляра (например, после работы офлайн).

---

## 2.6. Раздел «Комплект»

Ваш основной инструмент контроля готовности отчётности.

1. **Выберите период** — переключайте EID при смене отчётного периода.
2. **Смотрите прогресс** — сколько форм заведено, сколько в черновике, сколько сдано.
3. **Незаведённые формы** — список шаблонов, для которых ещё нет экземпляра; ссылка ведёт в каталог.
4. **Завести пустые формы** — создать недостающие экземпляры одной кнопкой.

Рабочий контекст (ZID + EID) запоминается: новые формы из каталога автоматически привязываются к выбранному периоду.

---

## 2.7. Раздел «Каталог»

Полный список из 76 шаблонов форм корпоративной отчётности.

- **Поиск** по коду (например, `N01_1`) или названию.
- **Фильтр по разделу** (N01, N02, … ND).
- **Создать** — новый экземпляр формы в «Мои формы».
- **PDF** — просмотр образца бланка (если загружен на сервер).

Используйте каталог, когда нужна одна конкретная форма вне пакетного заведения комплекта.

---

## 2.8. Настройки

**Настройки** — значения по умолчанию для **новых** форм:
- код предприятия;
- текстовое поле «Организация»;
- даты периода;
- единица измерения (обычно «тыс.руб.»).

Нажмите **Сохранить настройки** после изменений.

> Это не справочник организаций. Ваша привязка к организации (ZID) задаётся администратором в учётной записи.

В блоке **Учётная запись** отображаются ваш логин, роль и организация; кнопка **Выйти**.

---

## 2.9. Типовые сценарии пользователя

### Ежедневная работа
1. Войти → **Мои формы**.
2. Открыть форму со статусом «Черновик».
3. Заполнить → **Сохранить**.
4. Перейти к следующей форме.

### Завершение отчётного периода
1. **Комплект** → выбрать период → убедиться, что все 76 форм заведены.
2. Заполнить оставшиеся черновики.
3. По каждой форме: **Проверить увязки** → исправить → **Сдать**.
4. **Комплект** → убедиться, что все формы в статусе «Сдано».

### Ошибка в сданной форме
Обратитесь к администратору. Самостоятельно сданную форму изменить нельзя.

### Работа с новым периодом
1. **Комплект** → выберите новый период (EID).
2. **Завести пустые формы** — если комплект ещё не создан.
3. Заполняйте формы как обычно.

## 2.10. Частые вопросы (пользователь)

**Почему я не вижу раздел «Сводка и импорт»?**  
Этот раздел доступен только администраторам. Пакетные проверки выполняет ЦО.

**Почему не могу выбрать другую организацию?**  
Ваша учётная запись привязана к одной организации. Это сделано для разграничения данных.

**Форма не сохраняется / ошибка доступа**  
Проверьте подключение к интернету. Если форма в статусе «Сдано» — редактирование заблокировано. Обратитесь к администратору.

**Куда делись мои данные после смены браузера?**  
Данные хранятся на сервере, не в браузере. Войдите под тем же логином с любого устройства.

**Что означают статусы?**  
- **Черновик** — можно редактировать.  
- **Сдано** — форма зафиксирована, ожидает приёмки.
