## Кнопки редактора формы

| Кнопка | Назначение | Результат |
|--------|------------|-----------|
| **Сохранить** | Записать изменения на сервер | «Сохранено ЧЧ:ММ:СС» |
| **Сдать форму** | Зафиксировать форму | Статус «Сдано», правки закрыты |
| **Вернуть в черновик** *(admin)* | Открыть сданную форму для правок | Статус «Черновик» |
| **Пересчёт** | Пересчитать итоговые ячейки | Итоги обновлены |
| **Автопересчёт** | Итоги при вводе без кнопки | Настройка в браузере |
| **Проверить форму** | Проверка увязок | Список ошибок или OK |
| **Проверить расшифровки** | Контроль контрагентов | Ошибки/предупреждения |
| **Сохранить PDF** | Печатная версия | Файл PDF |
| **Excel** | Выгрузка в таблицу | Файл .xlsx |
| **Экспорт JSON** | Резервная копия | Файл .json |
| **Импорт** | Загрузить JSON в форму | Данные заменены |
| **Сбросить данные** | Очистить до пустого шаблона | Все ячейки пустые |
| **Удалить** | Удалить экземпляр | Форма исчезает из списка |
| **Образец PDF** | Пустой бланк | PDF в новой вкладке |

## Кнопки «Мои формы»

| Кнопка | Результат |
|--------|-----------|
| **Открыть** | Редактор формы |
| **Удалить** | Одна форма удалена |
| **Удалить выбранные (N)** | N форм удалено |
| **Импорт JSON** | Новая форма из файла |
| **Выбрать все (N)** | Все видимые отмечены |
| **Снять выбор** | Чекбоксы сброшены |

## Кнопки «Комплект»

| Кнопка | Кто | Результат |
|--------|-----|-----------|
| **Создать организацию** | admin | Новый ZID |
| **Создать период** | admin | Новый EID |

## Глоссарий

| Термин | Пояснение |
|--------|-----------|
| **ZID** | Код организации в справочнике |
| **EID** | Код отчётного периода |
| **Комплект** | До 76 экземпляров форм для пары ZID + EID |
| **Шаблон** | Описание структуры формы из каталога (76 штук) |
| **Экземпляр** | Заполненная копия шаблона с вашими данными |
| **Увязка** | Правило: значения в ячейках должны соотноситься определённым образом |
| **Сальдо** | Перенос остатков между формами/периодами |
| **Агрегация** | Суммирование данных нескольких org или экземпляров |
| **Черновик** | Форма в работе, редактирование открыто |
| **Сдано** | Форма зафиксирована, редактирование закрыто |
