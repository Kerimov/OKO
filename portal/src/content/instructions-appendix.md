## Глоссарий

| Термин | Определение |
|--------|-------------|
| **ZID** | Идентификатор организации в справочнике |
| **EID** | Идентификатор отчётного периода |
| **Комплект** | Набор экземпляров форм для ZID+EID (до 76 шт.) |
| **Шаблон** | Тип формы в каталоге (N01_1 и т.д.) |
| **Экземпляр** | Конкретная сохранённая форма с данными |
| **Увязка** | Правило контрольного соотношения между ячейками |
| **Сальдо** | Перенос остатков между формами/периодами |
| **Агрегация** | Сведение данных дочерних org в сводную |
| **Рабочий контекст** | Текущие ZID и EID на странице «Комплект» |

---

*При обновлениях портала актуальная версия инструкции всегда доступна в разделе **Инструкция** бокового меню.*
