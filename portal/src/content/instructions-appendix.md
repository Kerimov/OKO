## Кнопки редактора формы

| Кнопка | Результат после нажатия |
|--------|-------------------------|
| **Сохранить** | Данные на сервере, строка «Сохранено …» |
| **Сдать форму** | Статус «Сдано», правки закрыты |
| **Вернуть в черновик** *(admin)* | Статус «Черновик», правки открыты |
| **Пересчёт** | Итоговые ячейки пересчитаны |
| **Проверить форму** | Список ошибок увязок или OK |
| **Проверить расшифровки** | Ошибки/предупреждения sp_rash |
| **Сохранить PDF** | Файл PDF скачан |
| **Excel** | Файл .xlsx скачан |
| **Экспорт JSON** | Резервная копия .json |
| **Импорт** | Данные из JSON в форму |
| **Сбросить данные** | Пустой шаблон |
| **Удалить** | Форма удалена |

---

## Кнопки «Мои формы»

| Кнопка | Результат |
|--------|-----------|
| **Открыть** | Редактор формы |
| **Удалить** | Одна форма удалена |
| **Удалить выбранные (N)** | N форм удалено |
| **Импорт JSON** | Новая форма из файла |
| **Выбрать все** | Все видимые отмечены |
| **Снять выбор** | Чекбоксы сброшены |

---

## Кнопки «Комплект»

| Кнопка | Кто | Результат |
|--------|-----|-----------|
| **Создать организацию** | admin | Новый ZID |
| **Создать период** | admin | Новый EID |
| **Завести пустые формы (комплект)** | все | До 76 экземпляров создано |
| **Мои формы** | все | Переход к списку |

---

## Кнопки «Сводка и импорт» (только admin)

| Кнопка | Результат |
|--------|-----------|
| **JSON (N)** | Архив комплекта |
| **Excel** | Excel комплекта |
| **Пересчитать все** | Пересчёт всех форм периода |
| **Проверить все формы** | Пакетная проверка увязок |
| **Перенести сальdo** | Данные в целевой форме |
| **Свести комплект** | Агрегация дочерних org |
| **Создать агрегированную форму** | Сумма 2+ экземпляров |
