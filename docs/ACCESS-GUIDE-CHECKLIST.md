## Сводка покрытия по руководству

| Блок руководства | Покрыто |
|------------------|---------|
| Запуск / хранение данных | иначе (целевой стек), OK для прома |
| Завести пустые формы | да |
| Двухэтапное сальдо | портал да; десктоп нет UI |
| Заполнение + расшифровки + увязки | да |
| Сохранить на диск → головная | ReportPackage / JSON (иначе формат) |
| Совместная работа (сеть) | да (Tauri + папка) |
| Принять частично / сравнение | нет / иначе |
| Справочники займов·НЗС / N99 | нет |
| Агрегация | портал да |
| Резерв / только чтение | partial / иначе |

**Итог:** по основному пользовательскому циклу руководства (завести → сальдо* → заполнить → проверить → сдать) — **да, с оговоркой что сальдо и агрегация живут в портале ЦО**, а совместное заполнение — в Tauri. Пункты Access про «Принять частично», N99, принять справочники филиалов, F1/F11 как в Access и ini-бэкапы — **не повторены 1:1**.
