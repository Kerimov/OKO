## Сводка покрытия по руководству

| Блок руководства | Покрыто |
|------------------|---------|
| Запуск / хранение данных | иначе (целевой стек), OK для прома |
| Завести пустые формы | да |
| Двухэтапное сальдо | портал да; десктоп Package да (колонки) |
| Заполнение + расшифровки + увязки | да |
| Сохранить на диск → головная | ReportPackage JSON/ZIP (иначе формат Access mdb) |
| Совместная работа (сеть) | да (Tauri + папка) |
| Принять частично / сравнение | да (формы + cell-diff на `/tools`) |
| Справочники займов·НЗС / N99 | да (Tools: loans-nzs + N99 CSV) |
| Агрегация | портал да |
| Резерв / только чтение | daily + restore + VACUUM; иначе статусы |

**Итог:** по основному пользовательскому циклу руководства (завести → сальдо* → заполнить → проверить → сдать) — **да, с оговоркой что сальдо и агрегация живут в портале ЦО**, а совместное заполнение — в Tauri. Пункты Access про F1/F11 как горячие клавиши и ini-бэкапы — **не повторены 1:1**; «Принять частично», займы/НЗС и N99 — в `/tools`.
